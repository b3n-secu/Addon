# Icon Placeholder

Für ein echtes Icon sollte hier eine PNG-Datei mit dem Namen "icon.png" platziert werden.

Empfohlene Eigenschaften:
- Format: PNG
- Größe: 256x256 oder 512x512 Pixel
- Transparenter Hintergrund
- Darstellung: Modbus-Symbol, Zahnräder oder PLC-Steuerung

Icon-Ideen:
- Modbus-Logo mit Home Assistant Farben
- Industrielle Steuerung/PLC-Symbol
- Vernetztes Gerät mit Konfigurationssymbol

Sie können Icons erstellen mit:
- Canva.com
- GIMP
- Inkscape
- Online Icon Generators
